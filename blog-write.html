📤 게시 방법 (3단계)

  1. 위에서 글 작성 → 게시 파일 저장 클릭 → blog-draft.json 다운로드 (다운로드 폴더에 저장돼도 됩니다)
  2. PowerShell에서 프로젝트 폴더로 이동 후 실행: 
    cd "D:\경락\커서\site setting\outcallsite\1"
python scripts/publish_blog.py --add blog-draft.json
python scripts/generate_pages.py
    ※ 파일이 다운로드 폴더에만 있어도 자동으로 찾습니다.
  3. GitHub에 올리기: 
    git add data/blog-posts.json blog.html blog/ css/blog.css sitemap.xml
git commit -m "블로그 글 추가"
git push origin master

또는 채팅에 글 내용을 보내주시면 대신 올려 드릴 수 있습니다.
